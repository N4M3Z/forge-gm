If a scene reward (feat, item, boon) has no canonical D&D counterpart, create a compendium entry in `Compendium/Homebrew/Feats/` or `Compendium/Homebrew/Items/`. The scene Rewards table links via `[[Feat Name]]` or `[[Item Name]]`.

Never embed full homebrew descriptions in scenes. The scene is a reference sheet, the compendium is the definition.

Rewards must have mechanical backing in table format:

```markdown
| Reward              | Type | Mechanic                        |
| ------------------- | ---- | ------------------------------- |
| [[Void Resilience]] | Feat | Advantage on saves vs. psychic  |
```
