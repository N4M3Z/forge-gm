Voice tags describe how an NPC sounds. Place them inline after the NPC name in `[!quote]` dialogue callouts:

```markdown
> [!quote] [[Monidan]] #voice/low #voice/gruff
> *"You've been walking for two years."*
```

Available voice tags: `#voice/low`, `#voice/high`, `#voice/gruff`, `#voice/breathy`, `#voice/fast`, `#voice/slow`, `#voice/underbite`.

Sensory tags mark description content by sense: `#sense/sight`, `#sense/sound`, `#sense/smell`, `#sense/touch`, `#sense/taste`. Use them in `[!hint]` description callouts to help the GM select which senses to emphasize when reading aloud.
