Every session prep document includes these five elements for at-the-table improvisation:

| Element                 | What it contains                                 |
| ----------------------- | ------------------------------------------------ |
| **Strong Start**        | Opening beat that drops players into action       |
| **Secrets & Clues**     | Checklist (`- [ ]`) of discoverable information  |
| **Fantastic Locations** | Named locations with one-line descriptions        |
| **NPCs**               | Name, role, voice tag, key line of dialogue       |
| **Monsters**           | Encounter name, challenge rating, notes           |

Secrets & Clues use unchecked task markers. Check them off during play to track what was revealed. Undelivered secrets carry forward to the next session.
