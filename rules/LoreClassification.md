Every compendium entry is tagged with exactly one lore origin:

| Tag              | Meaning                                    | Example                              |
| ---------------- | ------------------------------------------ | ------------------------------------ |
| `#lore/canon`    | Official D&D content, unmodified           | Astral Plane, Githyanki, Mind Flayer |
| `#lore/homebrew` | Original to this campaign                  | Custom NPCs, invented locations      |
| `#lore/adapted`  | Canon with campaign-specific modifications | Gith (if backstory differs)          |

When classifying, verify against authoritative sources. Adapted entries must note what differs from canon in the body. When unsure, use `#lore/unclassified` and flag for GM review.

Reference sources for verification:

| Entity Type | Source                                                   |
| ----------- | -------------------------------------------------------- |
| Monsters    | [D&D Beyond](https://www.dndbeyond.com/monsters)        |
| Spells      | [D&D Beyond](https://www.dndbeyond.com/spells)          |
| Items       | [D&D Beyond](https://www.dndbeyond.com/magic-items)     |
| Races       | [D&D Beyond](https://www.dndbeyond.com/species)         |
| Lore/Planes | [Forgotten Realms Wiki](https://forgottenrealms.fandom.com/) |
| SRD         | [D&D 5e SRD](https://www.dndbeyond.com/sources/dnd/free-rules) |

Place external links in a `## References` section at the bottom of compendium entries. Homebrew entries do not get external links.
